# Chatbot Ollama

Download ollama by following instructions at https://github.com/ollama/ollama?tab=readme-ov-file 

## Quickstart

To run and chat with [Llama 3.1](https://ollama.com/library/llama3.1):

```
ollama run llama3.1
```

## Python packages setup

```
pip install ollama
```

## Run chatbot

```
python chatbot.py
```
